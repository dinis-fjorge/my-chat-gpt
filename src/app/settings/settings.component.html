Summary/Memory Generation Prompts

These are the prompts used by the summarization and context memory functions to produce summaries, contexts, memories, select relevant keywords, etc. for the purpose of compressing a large conversation into 4096 tokens that can be fed into the API. These prompts are used in conjunction with either recent messages or specific text to be fed into a seperate AI API call to produce the desired output. These start out with default values that I've prompt engineered to be good enough, but honestly the outputs from these commands kind of suck sometimes, so if you have a desire to take up some prompt engineering of your own, tweak with these text fields, or even completely rewrite them, until you get a better outputs for summaries, memories, keyword selections, etc.
